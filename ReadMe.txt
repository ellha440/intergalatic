How to run the application
1) In the terminal go to the intergalactic folder.
2) The project will be built using ant, if you have not installed ant on your computer then please do so: https://ant.apache.org/ or search for: ant download
3) Download junit from the official site: http://junit.org/
4) Place the junit jars in thoughtworks/lib  folder
5) Write in terminal: ant compile
6) Write in terminal: ant run
7) Follow the instructions (Type the absolute path to the input file)
8) The output will be written in the terminal
